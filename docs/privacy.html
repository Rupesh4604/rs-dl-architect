Last Updated: May 9, 2026

1. What is rs-dl-architect?

rs-dl-architect is an open-source skill (knowledge module) for Claude AI that provides expert guidance on deep learning architectures for remote sensing and satellite imagery. It consists of markdown files containing instructions and reference materials.
